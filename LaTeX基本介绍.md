# ``\LaTeX`` 基本介绍

## ``\LaTeX`` 是什么

> ``\LaTeX`` - A document preparation system

上面这段话是 ``\LaTeX`` 官网的标题，顾名思义，``\LaTeX`` 是一套排版系统，可以完成论文、书籍甚至 PPT 的排版工作。

## 为什么要用 ``\LaTeX``

* 论文需要：很多 SCI 期刊推荐甚至只接收 ``\LaTeX`` 投稿，并且这些期刊会提供 ``\LaTeX`` 模板。
* 由于是纯文本格式，可以使用 git 等版本控制工具进行版本管理。
* 更简单，你没看错，``\LaTeX`` 更简单。
    * 格式控制更简单，虽然没有所见即所得，但是格式都由自己准确控制，相比而言，如果对 word 的使用不了解，那么可能会出现格式莫名奇妙被改了的问题。
    * 样式修改更简单，只需要修改很少的代码，即可以从一种期刊的投稿格式切换到另一个期刊的投稿格式。
    * 参考文献管理更简单，用 ``\LaTeX`` 结合 zotero 管理参考文献，你写论文就不再需要考虑参考文献格式的问题。
