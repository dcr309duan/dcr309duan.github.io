延迟初始化

  • 设置为可空
  • lateinit var
  • lateinit var + ::nameView.isInitialized

密封类 sealed class

  • 密封类是一种特殊的抽象类。
  • 密封类和其子类必须定义在同一个文件当中。
  • 对于密封类来说，子类的个数是有限的。

内联类 inline class

  • 内联类是对某一个类型的包装
  • 内联类是类似于 Java 装箱类型的一种类型
  • 编译器会尽可能使用被包装的类型进行优化
  • 内联类可以实现接口，不能继承类，也不能被继承
  • 主构造器必须有且仅有一个只读属性
  • 不能定义有 backing-field 的其他属性
  • 被包装类型必须不能是反省类型
  • 内联类不能定义为其他类的内部类
