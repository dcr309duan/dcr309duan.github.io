Kotlin 内置类型

kotlin vs java

  • kotlin 没有像 Java 一样的基本类型
  • 变量声明：val, var
  • 类型推导
  • long: L not l
  • 隐式类型转换：e.toLong()
  • 无符号类型：UInt
  • === 比较引用，== 比较值

数组

基本类型的数组，Kotlin 有专门的类型

  • int[] -> IntArray
  • Integer[] -> Array<Int>

数组的创建：

  • intArrayOf()
  • arrayOf()
  • IntArray(5){it + 1} -> [1, 2, 3, 4, 5]
  • in

数组遍历：

  • array.indices

区间

Kotlin 集合框架

  • 增加了不可变集合框架的接口
  • 复用 Java API 的所有实现类型
  • 提供了丰富易用的方法，例如 forEach/map/flatMap
  • 运算符级别的支持，简化集合框架的访问

kotlin 通过类型别名的方式，将集合类型映射到了 Java 类。这样做的目的，是为了跨平台。

集合框架的读写：

  • stringList += "num: 3" 等价于 stringList.add("num: 3")
  • stringList -= "num: 3" 等价于 remove
  • stringList[5] = "Hello World" 等价于 set
  • val a = stringList[5] 等价于 get
  • map["Hello"] = 10
  • println(map["Hello"])

函数

  • Kotlin 中，函数也是一种类型，函数是 “一等公民”
  • 可以赋值、传递

函数定义

  • 关键字 fun
  • 函数名
  • 参数列表
  • 返回值类型（可类型推导）
  • 方法和函数的区别：方法有一个 receiver

函数的类型

  • () -> Unit
  • (Int) -> String
  • Foo.(String, Long) -> Any ==z (Foo, String, Long) -> Any

函数的引用

  • ::foo
  • Foo::bar
  • val f = ::foo
  • val g = ::foo
  • val h = Foo::bar
