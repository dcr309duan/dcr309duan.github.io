Kotlin 函数进阶

高阶函数

高阶函数就是，参数为函数hove返回值为函数的函数，可以理解为函数的函数（类似于数学中函数的高阶的概念）

函数引用：::println

  • Lambda 作为最后一个函数参数，可以将大括号放在外面
  • 只有一个 Lambda 表达式作为参数，可省略小括号

内联函数

forEach 函数的定义是使用 inline 关键字的，定义为乐内联函数。

内联函数是为了提升性能。

local return

non-local return

使用 crossinline 禁止 non-local inline

如果使用 noninline 禁止函数参数被内联

内联属性

  • 没有 backing field 属性的 getter/setter 可以被内联

内联函数的限制

  • public/protected 的内联方法只能访问对应类的 public 成员
  • 内联函数的内联函数参数不能被存储
  • 内联函数的内联函数参数只能传递给其他内联函数参数
