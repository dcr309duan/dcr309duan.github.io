## üö¶ Getting Started

### Prerequisites

- Ruby
- Bundler
- Jekyll

### Installation

1. **Clone the repo**
   ```bash
   git clone https://github.com/dcr309duan/dcr309duan.github.io.git
   cd dcr309duan.github.io
   ```

2. **Install dependencies**
   ```bash
   bundle install
   ```

3. **Run locally**
   ```bash
   bundle exec jekyll serve
   ```
